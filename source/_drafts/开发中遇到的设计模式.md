在学习多线程的时候，我看过的很多资料都会强调创建线程有好几种方式。

其中，最简单的创建线程的两种方式就是继承Thread的子类重写run方法，另外。

要理解这两种创建线程的方式有什么不同，知识要

其实原因很简单，就是因为Thread类提供了几种不同的构造器。

```
@Override
public void run() {
    if (target != null) {  //调用组合的target对象的run方法
        target.run();
    }
}
```
